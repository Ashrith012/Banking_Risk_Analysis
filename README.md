# 🏦 Banking Risk Analytics Dashboard

## 📌 Overview

The Banking Risk Analytics Dashboard is a Power BI project developed to analyze banking risk factors, customer profiles, and loan-related metrics. The dashboard helps financial institutions monitor risk exposure, evaluate customer creditworthiness, and make informed lending decisions through interactive data visualizations.

---

## 🎯 Project Objectives

- Analyze customer financial profiles.
- Identify high-risk and low-risk customer segments.
- Monitor loan distribution and approval trends.
- Evaluate factors affecting banking risk.
- Support data-driven decision-making in credit and lending operations.

---

## 🛠️ Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📊 Dashboard Features

### Executive Summary
Provides an overview of:

- Total Customers
- Total Loan Amount
- Risk Categories
- Loan Approval Metrics
- Customer Segmentation
- Financial Performance Indicators

### Risk Analysis
- Customer risk classification
- Creditworthiness analysis
- High-risk customer identification
- Risk distribution across customer groups

### Loan Analytics
- Loan approval trends
- Loan amount distribution
- Customer borrowing patterns
- Lending performance insights

### Interactive Filtering
Users can dynamically analyze data using:

- Customer Segments
- Risk Categories
- Income Levels
- Loan Types
- Geographic Filters

1. Download the `.pbix` file from this repository.
2. Open it using Microsoft Power BI Desktop.
3. Refresh the dataset if required.
4. Explore the dashboard using filters and slicers.
5. Analyze risk and lending insights through interactive visualizations.
